# Proje Durumu: Autochess Hybrid

## Proje Özeti

**Autochess Hybrid**, hex-grid tabanlı otomatik savaş (autochess) oyun motoru simülasyonudur.

### Temel Özellikler
- 8 oyunculu lobide 101 kartlık havuzdan stratejik kart seçimi
- Hex-grid tabanlı (radius 3, 37 hex) combat sistemi
- Pasif yetenek sistemi (synergy, combo, combat passives)
- AI stratejileri ve KPI takibi

### Mevcut Durum

| Bileşen | Durum |
|---------|-------|
| `engine_core/` | Tamamlandı - Çekirdek oyun motoru |
| `v2/` (Pygame UI) | Geliştirilmekte - Scene-based mimari |
| Test suite | Aktif - Kapsamlı birim testleri |

### Teknoloji Stack'i

| Katman | Seçim |
|--------|-------|
| Language | Python 3.11+ |
| Rendering | Pygame-CE 2.4+ |
| Window | 1920×1080 @ 60 FPS |
| Hex koordinat | Axial (q, r) |

### Dizin Yapısı

```
hybrid/
├── engine_core/     # Oyun motoru
├── v2/              # Pygame UI
│   ├── core/        # GameState, SceneManager
│   ├── scenes/      # Lobby, Shop, Combat, GameOver
│   └── ui/          # Panel, HexGrid, Widgets
├── tests/           # Test suite
├── tools/           # Geliştirici araçları
└── assets/data/    # cards.json, passives.txt
```
